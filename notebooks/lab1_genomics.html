Shishi Luo¶

To practice some basic Tables operations, we'll play with data from the NIH NCBI database that contains the genome size, number of genes, number of proteins, etc, for tens of thousands of organisms

Data source¶

http://www.ncbi.nlm.nih.gov/genome/browse/ has an extensive list of genome information by organism.

You can download the information about whatever organisms you're interested in. Here's an example of the data I've downloaded for some model organisms.

Simply counting the size of the genome or the number of genes does not always tell us much about what kind of organism it belongs to.

Finally, recalling the central dogma of molecular biology, let's plot the number of genes in an organism against the number of proteins.

In [18]:
model_species.scatter('Genes', 'Proteins')

Does a similar relationship exist between the size of the genome and the number of genes?

Now that we have such a long list, use a histogram to visualize the distribution of genome size.

In [26]:
animals.hist('Size (Mb)', bins=20, normed=False)

Note the units for genome size are in millions of bases (Mega bases, Mb for short). Why didn't we do this earlier?

Let's use the information in the column 'SubGroup' to see if the sizes break down by sub-group

In [44]:
animals.pivot_hist('SubGroup', 'Size (Mb)', bins=np.linspace(1,4000, 10), 
                   normed=False, orientation='horizontal')
