Import the data¶

Our data will come from LANL HIV database. This time, we use the intra-patient search to find sequences that come from the same patient. The data we use today come from a study by Archary et al. The dataset contains env sequences from four different patients. The patient identifier of four of the sequences has been replaced with 'X1', 'X2', 'X3', and 'X4'.

Your task is to see if you can identify which patient each of these mystery sequences came from.

Distances between sequences¶

Let us assume that sequences from the same patient are more similar than sequences from different patients. Then, to identify which patients the mystery sequences are from, we need a way to measure the distance between sequences.

One approach is to use Hamming Distance. The Hamming Distance between two strings (of equal length) is the number of positions where the strings differ.

For example, 'Hello, Dolly!' and 'Help-, Dolly!' have Hamming Distance 2.

The function below outputs the Hamming Distance between two strings.

In [3]:
def hammingDistance(s1, s2):
    '''For two strings s1 and s2, output the number of position that they differ in.
    Strings s1 and s2 must be the same length.'''
    if len(s1)!=len(s2):
        return('The input strings are not of the same length.')
    else:
        return(sum([i!=j for i,j in zip(s1, s2)]))

# Creating a function that takes as input an unknown individual and returns the Patient ID
# to whom the sequence most likely belongs to

def whose_is_it(x):
    x_sequence = intrapatient_seqs.where(intrapatient_seqs.column(0), x).column(1).item(0)
    hdist_x = []
    for seq in intrapatient_seqs.column(1):
        hdist_x += [hammingDistance(seq, x_sequence)]
    patient_id_x = intrapatient_seqs.with_column('Hamming Distance', hdist_x).sort(2).column(0).item(1)
    print('Sequence', x, 'most likely belongs to the individual with patient ID', patient_id_x)

def plot_distances(x):
