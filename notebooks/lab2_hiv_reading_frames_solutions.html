def isStop(string):
    '''returns 1 if string (of length 3) is a stop codon'''
    if len(string)!=3:
        return('String needs to be of length 3!')
        
    else:
        return(int(string.upper() in ['TAG', 'TAA', 'TGA']))

Scanning the full genome¶

We are now ready to scan the entire HIV genome for start and stop codons. All that's needed is some arithmetic and a for loop.

DNA is translated in triplets (called codons). For the reading frame beginning at the first base, hxb2[:3] returns the first codon, hxb2[3:6] the second codon, etc.

Every DNA sequence has three reading frames (positions from which codons can be read). So far, we have plotted the start and stop codons for the reading frame beginning with the first base. The reading frames beginning at the second and third base could also include stop and start codons.

Q12: Make a plot for the start codons and stop codons for each reading frame.

In [11]:
for offset in range(3): # offset=0 is the first reading frame, offset=1 the second, etc
    stop_codons=[]
    start_codons=[]
    for i in range(n_codons-3-offset):
        stop_codons.append(isStop(hxb2[i*3+offset:i*3+offset+3]))
        start_codons.append(isStart(hxb2[i*3+offset:i*3+offset+3]))
    plots.figure(figsize=(20,1))
    plots.plot(stop_codons, 'r')
    plots.xlim([0,n_codons])
    plots.ylim([0,1.1])
    plots.figure(figsize=(20,1))
    plots.plot(start_codons, 'g')
    plots.xlim([0,n_codons])
    plots.ylim([0,1.1])

For the perfectionists¶

Ideally, we'd like a vertical bar for each start and stop codon. In the plots above, the 'vertical' bars are actually really squished triangles (a line connecting 0, 1, and 0 very tightly).

Try typing plots.axvline(x=5, linewidth=.5, color='r') to see how one can plot vertical bars.
